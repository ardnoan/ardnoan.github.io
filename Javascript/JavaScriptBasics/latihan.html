Mengekstrak String

Kita dapat mengekstrak substring dari string menggunakan metode berikutnya
  • Irisan (mulai, akhir)
  • substring (awal, akhir)
  • substri (awal, panjang)

Metode Irisan

metode slice mengekstrak bagian dari string dan mengembalikannya sebagian string baru. tu tidak mengubah string asli. Dibutuhkan dua parameter numerik indes awal (posisi) indeks akhir.



Metode slice dapat mengambil nilai negatif, nilai negatif mulai dihitung dari akhir string



Metode substring

Metode substring bekerja dengan cara yang sama seperti metode slice.



Satu-satunya perbedaan antara metode slice dan substring adalah bahwa metode substring tidak dapat mengambil nilai negatif.

Metode substr

Metode substr mengekstrak bagian dari string, mulai dari indeks yang ditentukan dan diperpanjang untuk panjang tertentu sesudahnya. Dibutuhkan dua parameter numerik indeks awal, panjang jumlah karakter yang akan diekstrak.

Menghilangkan nilai kedua

Jika anda menghilangkan nilai kedua dari slice, substring dan substr mereka akan mengembalikan sisa string.

Mengganti string didalam string

Metode replace menggantikan teks yang ditentukan didalam string dan mengembalikan string baru. Itu tidak mengubah string asli, dibutuhkan dua prameter string untuk diganti, string pengganti.

Perhatikan bahwa hanya kemunculan pertama pertama 'study' yang diubah, untuk mengganti semua string cocok gunakan ekspresi dengan itu /g bendera.

Sekarang semua kemunculan 'study' digantikan oleh 'everyone'. Catatan! metode penggantian memiliki casing yang sensitif, untuk mengganti teks secara tidak sensitif, gunakan tanda /i dalam ekpresi reguler.



Seperti yang diperhatikan sekali lagi, itu hanya menggantikan kejadian pertama, untuk mengganti semua secara tidak sensitif, gabungkan tanda /g dalama ekpresi reguler.
