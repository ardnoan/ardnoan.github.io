Cascading Style Sheet (CSS)

  1. CSS adalah singkatan dari Cascading Style Sheet.
  2. CSS adalah bahasa stylesheet yang digunakan untuk menggambarkan presentasi dokumen HTML.
  3. CSS Menjelaskan bagaimana elemen harus ditampilkan dilayar.

Prasyarat CSS

Pemahaman dasar tentang HTML, untuk mulai belajar CSS harus memiliki pemahaman dasar tentang HTML terlebih dahulu.

Mengapa Menggunakan CSS

CSS mendefinisikan gaya halaman web termasuk desain, tata letak, dan variasi tampilan diseluruh perangkat dan ukuran layar.

CSS dapat menghemat banyak pekerjaan. Lembar gaya eksternal disimpan sebagai " .css " dapat digunakan untuk menata beberapa halaman web sekaligus!.

Input

                body {
                    background-color: blue;
                }
                p {
                    font-size: 14px;
                    color: aqua;
                }

Output

backgroundnya warna blue

Text warna aqua

ukuran text 14px

CSS sangat menyenangkan dan seru untuk dipelajari terutama dipelajaran berikutnya, teruslah belajar, maju dan nikmati prosesnya!.
