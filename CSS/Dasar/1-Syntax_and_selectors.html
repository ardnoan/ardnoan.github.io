CSS Syntax

Aturan CSS terdiri dari selector dan declaration memblokir.

div {

background-color: black;

width: 200px;

height: 200px;

}

Selector atau sekelompok selector menargetkan elements yang kita inginkan untuk style.

div

{

background-color: black;

width: 200px;

height: 200px;

}

Declaration blocks dikelompokan dalam blok. Setiap blok deklarasi berisi deklarasi CSS yang dibungkam oleh kurung kurawal buka { dan kurung kurawal tutup }

div {

background-color: black;

width: 200px;

height: 200px;

}

Declaration atau declaration CSS terkandung di dalam blok deklarasi. Setiap deklarasi harus dipisahkan dengan titik koma; jika tidak, kode tidak akan berfungsi. Diperbolehkan untuk tidak menempatkan titik koma ti deklarasi terakhir.

div {

background-color: black;

width: 200px;

height: 200px;

}

Declaration CSS terdiri dari sepasang properti dan nilai yang dipisahkan oleh titik dua : .

background-color : black;

border dengan warna merah itu namanya property, border berwarna biru namanya titik dua yang memisahkan dua entitas, dan yang terakhir yang berwarna hijau adalah value atau nilai.

  1. properti: pengidentifikasi yang dapat dibaca manusia yang menunjukan fitur style mana yang akan diubah;
  2. value: itu diberikan ke properti yang menunnjukan bagaimana fitur style yang ditentukan harus diubah.

Di CSS ada lebih dari 300 properti berbeda dan nilai berbeda dalam jumlah hampir tak terbatas. Tidak semua pasangan properti dannilai diperbolehkan. Setiap properti memiliki daftar nilai valid tertentu.

Properti maupun nilai peka huruf besar-kecil di CSS. hati-hati jika code tidak berjalan mungkin itu salah huruf besar dan kecilnya.
