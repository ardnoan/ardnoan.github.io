Formulir HTML

Formulir HTML dapat digunakan untuk mengumpulkan data pengguna, membuat formulir login, formulir pendaftaran, formulir pencarian, formulir kontak dan sejenisnya.

Element <form> mendefinisikan Formulir HTML

Attribut Formulir HTML

  1. accept-charset: mendefinisikan set karakter dari form;
  2. action: spesies halaman form-handler/ page-handler, biasanya halaman .php disisi server tempat data diproses;
  3. autocomplete :memberi tahu browser untuk secara otomatis element formulir lengkap;
  4. enctype: menentukan pengkodean data yang dikirimkan oleh formulir;
  5. method : menentuka metode HTTP untuk digunakan saat mengirimkan formulir, misalnya GET atau POST;
  6. name, id, class : digunakan untuk mengidentifikasi formulir, untuk tujuan penataan dan manipulasi objek dokumen;
  7. novalidate : memberi tahu browser untuk tidak menvalidasi formulir;
  8. onSubmit : digunakan untuk memmanggil fungsi javascrip misalnya untuk bentuk validasi saat formulir dikirimkan;
  9. target : menentukan target formulir misalnya target="_blank", form-handler akan dibuka di new tab.

Untuk memngirimkan formulir HTML pengguna harus mengklik tombol "kirim".

Pengguna akan dialihkan ke halaman form-handler yang ditentukan oleh attribut tindakan tempat data input diproses.

form-handler biasanya adalah halaman server Hypertext Processor (PHP) yang menjalankan skrip untuk memproses data masukan.

Untuk saat ini, hanya akan menggunakan halaman "sampel-page-handler.html" sebagai form-handler dan berpura-pura seolah-olah data yang dikirimkan diproses disisi server.

Contoh

<form action="sample-page-handler.html" method="GET">

<input type="text" name="fullname" placeholder="Full name"><br>

<input type="text" name="age" placeholder="Age"><br>

<input type="text" name="favourite_food" placeholder="favourite food"><br>

<input type="text" name="hobby" placeholder="hobby"><br>

<input type="submit">

</input>

Metode attribut

Attribut metode menenentukan metode HTTP mana yang akan digunakan kapan mengirimkan input data ke halaman form-handler itu bisa berupa GET atau POST.

Metode GET vs POST HTTP

HTTP Methods
GET POST
Nilai default Bukan nilai default
Kurang aman Lebih aman
Data ditulis di URL alaman Data tidak tertulis di alamant URL
Lebih rentan terhadap peretasan Kurang rentant terhadap peretasan
Tidak boleh digunakan saat mengirimkan data pribadi Harus digunakan untuk mengirimkan data pribadi.

Element Formulir HTML

Element formulir HTML adalah element di dalam element <form> kamu akan mempelajari berbagai jenis element formulir HTML termasuk input, kotak teks, tombol, kotak ceklis, daftar dropwdown, dan lainnya di pelajaran berikutnya.

Contoh

<form>

<!-- HTMLf orm element will be here -->

</form>

Element Masukan

Element input adalah bidang input di dalam formulir. ada berbagai jenis bidang input dan kamu akan mempelajarinya masing-masing dipelajaran berikutnya.

Nama attribut

Setiap element form kecuali tombol "kirim" harus memiliki attribut nama uniknya sendiri kecuali data tidak akan diserahkan ke form-handler karena digunakan sebagai referensi untuk mendapatkan nilai dari setiap element yang dimasukan oleh pengguna

Pelajaran berikutnya dalah semua tentang element formulir HTML lanjutkan kemudian nikmati dan rasakan.
