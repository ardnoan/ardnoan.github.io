Gaya HTML

Gaya HTML digunakan untuk gaya element HTML itu juga berarti mengubah nilai default.

Misalnya, penataan gaya dapat mengubah nilai default warna teks sebagai hitam, warna latar belakang putih, peratan teks sebagai kiri dan ukuran teks sebagai 12 piksel.

Lembar Gaya Internal (gaya internal)

Lembar gaya internal terdiri dari satu atau lebih kumpulan aturan Cascading Style Sheet (CSS).

Kumpulan aturan CSS terdiri dari pemilih dan blok deklarasi yang dikelilingi oleh kurung kurawal yang berisi satu atau lebih deklarasi CSS yang dipisahkan oleh titik koma.

Setiap deklarasi menyertakan nama properti CSS dan nilain, dipisahkan oleh titik dua.

Semuanya diapit didalam element <style> dengan atribute type="text/css" yang disertakan didalam element <head>.

Sintaks Lembar Gaya Internal

                <style type="text/css">
                    p {
                        font-size: 14px;
                    }
                </style>

Gaya Inline

Gaya sebaris digunakan untuk menata element menggunakan attribut gaya dengan deklarasi CSS didalamnya yang mirif dengan gaya internal.

Sintaks Styling Inline

<div style="property: value; property: value;">

Contoh Warna Latar Belakang

                <!DOCTYPE html>
                <html>
                <head>
                    <title> Coba Sendiri </title>
                </head>
                <body style="background-color: gold">
                    <!-- Isi Disini -->
                </body>
                </html>

Output

Contoh Warna teks

                <p style="color: blue"> Warna saya biru </p>
                <p style="color: red"> Warna saya merah </p>
                <p style="color: green"> Warna saya hijau </p>
                <p style="color: yellow"> Warna saya kuning </p>
                <p style="color: pink"> Warna saya pink </p>

Output

Warna saya biru

Warna saya merah

Warna saya hijau

Warna saya Kuning

Warna saya pink

Contoh font-family teks:

<h1 style="font-family: Times New Roman"> saya Times new roman </h1>

Output

saya Times new roman

Contoh Ukuran Teks

<p style="font-size: 25px;">Ukuran saya 25 piksel>

<p style="font-size: 50px;">Ukuran saya 50 piksel>

Output

Ukuran saya 25 piksel

Ukuran saya 50 piksel

Contoh perataan teks

<p style="text-algin: left;"> Saya rata kiri

<p style="text-align: center;"> Saya rata tengah

<p style="text-algin: right;"> Saya rata kanan

Output

Saya rata kiri

Saya rata tengah

Saya rata kanan
