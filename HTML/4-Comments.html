Komentar HTML

Komentar HTML adalah teks, frasa atau kalimat didalam HTML yang tidak akan ditampilkan oleh browser.

Mereka hanya ditampilkan dalam kode dan tidak dirender oleh browser.

Mengapa menggunakan komentar HTML

Komentar HTML membantu pengembangan web pemula dan berpengalaman untuk mengatur kode mereka dengan mudah.

Mereka bertindak seperti catatan tempel di file HTML.

Bagaimana cara menulis komentar HTML?

Komentar HTML dimulai dengan <!-- dan diakhiri dengan --> Ini terlihat seperti ini.

<!-- Komentar ada disini -->

Contoh:

<p>

Kalimat yang ditampilkan dibrowser.

<!-- Komentar satu baris -->

</p>

</p>


<p>

Kalimat yang ditampilkan dibrowser.

<!-- Ini adalah komentar multi-baris

bebrapa frasa masuk ke Sinii -->

</p>


<div> <!-- Meletakan komentar disamping tag awal -->

Kalimat, Element HTML, dan gambar, dll

</div> <!-- Menenmpatkan komentar disamping tag akhir -->

Output

Kalimat yang ditampilkan dibrowser.

Kalimat yang ditampilkan dibrowser.

Kalimat, Element HTML, dan gambar, dll

Pada contoh yang diberikan diatas, kamu telah belajar bahwa kamu dapat membuat satu baris dan multi-baris komentar HTML.

Dan Jika kamu diperhatikan kembali, kami memberi komentar disamping tag awal dan tag akhir.

Itu agar kita dengan mudah mengenali dimana element tertentu dimulai dan berakhir.
