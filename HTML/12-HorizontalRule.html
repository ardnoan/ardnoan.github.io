HTML Horizontal Rule

Aturan Horizontal HTML merender garis dibrowser web. Ini adalah sebuah element kosong.

Element <hr/> mendefinisikan Aturan Horizontal HTML.

Lihatlah contoh dibawah ini, untuk melihat cara kerjanya.

Contoh

<p> ini adalah sebuah paragraf. </p>

<hr/>

< Ini adalah sebuah paragraf.

Output

ini adalah sebuah paragraf.

Ini adalah sebuah paragraf.

Menata dan mengubah warna aturan horizontal HTML

Pada contoh dibawah ini kamu akan belajar begaimana mengatur gaya HTML horizontal aturan dengan mengubah lebar dan tingginya.

Untuk mengubah warnanya kita perlu menggunakan attribut color.

Contoh

<!-- lebar -->

<hr style="width: 50%; color="pink"/>

<!-- tinggi -->

<hr style="height: 40px; color="pink/>

<!-- color -->

<hr style="color: gold"/>
