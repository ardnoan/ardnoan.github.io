Input HTML5

Element <input> adalah element bentuk.

Dalam pelajaran kali ini kita tidak akan memasukan semua tipe input yang berbeda ke dalam element <form> karena kita hanya akan menggunakan javascript untuk mendapatkannya.

Namun, dalam menggunakan Hypertext Pocessor (PHP), element masukan page handler/formhandler harus berada di dalam element <form> dan attribut namanya adalah wajib.

Element <input> adalah element kosong dan sebaris.

Dengan Javascript kita dapat dengan mudah mendapatkan nilai dari tipe <input> berbeda yang ditentukan oleh attribut type.

Teks

Kolom teks baris tunggal tempat jeda baris diabaikan

<input type="text" id="demo">

<br>

<button onclick="ambilNilai()"> Ambil nilai </button>

<script type="text/javascript>

function ambilNilai() {

alert(document.getElementById(demo).value);

}

</script>

File

Dapat menentukan atribut accept untuk hanya mengizinkan jenis file tertentu, misalnya. <input type="file" accept-".pdf">.

Attribut accept mungkin tidak berfungsi di berbagai perangkat.

<input type="file" id="demo" onchange="readFile();">

<!-- Kamu harus memilih file gambar agar demo ini berfungsi -->

<img src="" alt="File yang dipilih akan tampil disini" id="result">

<script type="text/javascript>

function readFile() {

var reader = new FileReader();

var file = document.getElementById('demo').file[0];

reader.onload = function(e) {

document.getElementById('result').src = e.target.result;

}

reader.readAsDataURL(file);

}
