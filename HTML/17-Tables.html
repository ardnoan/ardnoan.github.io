Tables HTML

Table HTML mewakili data dalam bentuk table. itu terdiri dari kolom dan baris yang berisi data.

Element <table> mendefinisikan tabel HTML

Element table HTML

  1. <thead> Mendefinisikan header tabel;
  2. <tbody> mendefinisikan isi tabel;
  3. <tfoot> mendefinisikan footer tabel;
  4. <tr> mendefinisikan baris tabel;
  5. <th> mendefinisikan judul tabel;
  6. <td> mendefinisikan data/sel tabel;
  7. <colgroup> menentukan grup dari satu atau lebih kolom dalam tabel;
  8. <col> menentukan properti kolom untuk setiap kolom dalam tabel colorgrup (element kosong);
  9. <caption> mendefinisikan keterangan untuk tabel.

Attribut Tabel HTML

  1. align mendefinisikan perataan tabel;
  2. bgcolor menentukan warna latar belakang tabel;
  3. border menentukan ukurang bingkai yang mengelilingi tabel;
  4. cellpadding menentukan ruang antara konten sel dan batasnya;
  5. cellspacing menentukan jarak antara dua sel;
  6. frame menenetukan sisi mana dari bingkai yang mengelilinginya tabel harus ditampilkan;
  7. rules menentukan dimana aturan harus muncul dalam dabel;
  8. sunmary mendefinisikan teks alternatif yang merangkum isi Tabel;
  9. width menentukan lebar tabel.

Sebuah catatan! semua attribut tabel HTML yang disebutkan sudah tidak digunakan lagi, artinya attribut tersebut mungkin tidak lagi berfungsi dalam waktu dekat.

Meskipun attribut sudah tidak digunakan lagi, kita masih bisa membuat efek yang mirip dengan menggunakan gaya atau Cascading StyleSheet (CSS)

Contoh Sederhana :

                <table>
                    <thead>
                        <tr>
                            <th> first name </th>
                            <th> last name </th>
                        </tr>
                    </thead>
                    <tbody>
                        <tr>
                            <td> Asep </td>
                            <td> Ahmad </td>
                        </tr>
                    </tbody>
                    <tfoot>
                        <tr>
                            <td> footer </td>
                            <td> footer </td>
                        </tr>
                    </tfoot>
                </table>

Output

first name last name
Asep Ahmad
footer footer

Menciutkan perbatasan tabel HTML

Untuk menciutkan batas tabel menjadi satu batas, kita harus menyetel properti gaya border-collapse "dari tabel ke nilai" collapse "i.e" border-collapse: collapse".

Contoh perbatasan dan colspan yang diciutkan tabel:

                <style type="text/css">
                    table, th, td {
                        border-collapse: collapse;
                        border: 1px solid black;
                    }
                </style>
                <table>
                    <thead>
                        <tr>
                            <th colspan="3">henshin</th>
                        </tr>
                        <tr>
                            <th>first name</th>
                            <th>last name</th>
                            <th>nationality</th>
                        </tr>
                    </thead>
                    <tbody>
                        <tr>
                            <td>asep</td>
                            <td>ahmad</td>
                            <td>asura</td>
                        </tr>
                        <tr>
                            <td>maulana</td>
                            <td>saepudin</td>
                            <td>uzumaki</td>
                        </tr>    
                    </tbody>
                </table>
