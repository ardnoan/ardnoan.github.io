HTML links

Links HTML sangat diperlukan untuk semua halaman web. Semua situs web memiliki links. Links HTML adalah hyperlink.

Link membantu user melompat dari halaman web mereka saat ini ke web lain.

Untuk membuat links HTML kita hanya perlu menggunakana element <a> dengan attribut href yang menentukan URL ke suatu halaman.

Stukrut link HTML

  1. Mulai tag <a>.
  2. Attribut href. misalnya href="indeks.html".
  3. Konten element. SEbuah teks misalnya Halaman berada itu juga bisa berupa gambar dari element HTML lainnya.
  4. Tanda akhir </a>.

Contoh Links Internal

Menghubungkan dalam direktori kami

<a href="lokasifile.html">contoh

Contoh Penatuan Eksternal

Menghubungkan diluar direktori kami.

<a href="https://www.wikipedia.org">Wikipedia</a>

Membuat tautan di tab baru contoh

Kita dapat dengan mudah membuka tautan di tab baru.

Untuk melakukannya kita perlu menggunakan attribut target dengan nilai _blank.

Ini akan sangat membantu jika kamu tidak ingin pengunjung meninggalkan situs web kamu.

<a href="https://www.wikipedia.org" target="_blank">
