Konten HTML dapat diedit

Kita dapat membuat element HTML menjadi dapat diedit dengan menentukan.

Attribut kontent dapat diedit

Attribut contenteditable adalah attribut global yang dapat memiliki dua kemungkinan nilai.

  1. contenteditable="true" menunjukan bahwa HTML element harus dapat diedit;
  2. contenteditable="false" menunjukan bahwa element HTML tidak boleh diedit.

Attribut contenteditable tidak memerlukan nilai jika ada di tag awal element HTML.

Contoh

<p contenteditable>bisa diedit teks</p>

Output

bisa diedit teks

Styling editable

<p style="caret-color: red;">merah</p>

output

merah

Removing outline

<p style="outline: none;">menghilangkan outline</p>

output

menghilangkan outline
