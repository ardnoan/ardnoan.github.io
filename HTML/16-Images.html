Gambar HTML

Gambar HTML memang diperlukan atau diperlukan untuk setiap situs web. Gambar membantu situs web menjadi lebih menarik bagi pengunjung.

Bayangkan sebuah halamann web tanpa satu gambarpun apakah kamu masih akan menjelajahinya? tentu saja tidak akan menjelajahinya kemungkinan akan langsung keluar dari web tersebut.

Untuk meletakan gambar di situs web kami, kami hanya perlu menggunakan element <img/> dengan attribut src untuk menentukan URL atau lokasi gambar.

<img> adalah element kosong dan sebaris.

Attribute src

Kami menggunakan attribut src untuk menentukan URL gambar atau jalur file.

Contoh :

<img src="gambar1.png"/>

Attribut alt

Terkadang gambar mungkin tidak dimuat di browser pengguna karena koneksi internet yang lambat, server yang lambat, gambar dihapus dari direktori atau nilai URL yang salah ditentukan dalam attribut src.

Attribut alt menyediakan teks alternatif untuk gambar yang tidak sampai kebuka atau ke load gambarnya.

Contoh :

<img src="gambar.png" alt="saya adalah alternatif">

Pada contoh diatas URL gambar yang diberikan tidak benar-benar ada. Oleh karena itu, teks alternatif ditampilkan sebagai gantinya. Nilai biasanya dalam piksel.

Contoh :

<img src="contoh.png" width="350" height="400"/>

Untuk mengikuti praktik terbaik, kita dapat menggunakan attribut style untuk mengubah ukuran gambar dengan lebar dan tinggi dan nilainya juga bisa dalam piksel.

Contoh :

<img src="gambar.png" style="float: kanan"/>

Seperti yang kamu lihat hasilnya sama dengan menggunakan attribut <width> dan <height> tetapi keuntungan menggunakan attribut <style> adalah lebar dan tingginya tidak akan ditimpa oleh stylesheet.

Gambar Mengambang (kiri atau kanan)

Kita dapat mengapungkan gambar ke sisi kiri atau kanan teks. Untuk mencapai itu kita perlu menggunakan attribut style.

Dengan properti float CSS. Dan nilai kiri atau kanan. Berikut adalah pelajaran untuk HTML styling.

Contoh :

<p>

<img src="gambar.png" style="float: left;"/>

</p>

<p>

<img src="gambar.pnh" style="float: right;"/>

</p>

Gambar sebagai tautan

Ada beberapa situasi dimana kita membutuhkan gambar untuk bertindak sebagai tautan.

Untuk melakukannya cukup lampirkan gambar dengan element <a> dengan attribut href-nya.

Contoh :

<a href="https://www.example.com">

<img src="gambar.png"/>

</a>

Pada contoh yang diberikan diatas gambar yang digunakan ada di direktori/ folder yang sama dengan file HTML.

Gambar di folder lain

Biasanya kami menyimpan semua gambar kami dalam sub-folder untuk menjaga direktori kami tetap bersih dan tertata dengan baik.

Untuk menentukan gambar dari folder, lihat contoh dibawah ini :

Contoh :

<!-- didalam folder di direktori root -->

<img src="/foto/gambar.png"/>

<!-- dari folder yang sama tempat file HTML berada

<img src="gambar.png"

<!-- Dalam folder -->

<img src="foto/gambar.png"/>

Nilai dari attribut src disebut file path, berikut adalah pelajaran untuk itu.

Gambar Dari server eksternal

Terkadang kita perlu memasukan gambar dari situs web lain seperti facebook, google, atau imgur ke situs web kita.

Inilah cara melakukannya.

Contoh :

<img src="http://www.example.com/images.png"/>

tip

  1. Kamu dapat menggunakan semua jenis gambar yang kamu inginkan. misalnya jika nda menginginkan gambar animasi maka gunakan .gif
  2. Jangan menaruh banyak gambar dihalaman web. Halaman kamu mungkin memuat teralu lambat dan pengunjung kamu mungkin meninggalkan situs dan tidak pernah kembali lagi.

HTML dan Gambar dipenyimpanan ponsel kamu

Jika kamu menyimpan kode contoh di atas dimana saja di penyimpanan ponsel kamu, gambar tidak akan ditampilkan.

Menagapa? karena gambar hanya disimpan didirektori aplikasi ini dan bukan di penyimpanan ponsel kamu.

Ikuti langkah-langkah ini untuk menyisipkan gambar dengan benar ke halaman web kamu saat kamu berada di penyimpanan ponsel.

  1. Buka file manajer apa saja;
  2. Buat contoh folder;
  3. Tempel gambar apapun ke folder itu. ingat dnama file gambar;
  4. simpan sebagai index.html ke folder yang tealh kamu buat. untuk menyimpan itu.
