Video HTML

Untuk menyematkan video dalam HTML, kita harus menggunakan element <video> dengan attribut src atau element turunan <source> dengan attribut src dan type untuk masing-masing.

Element <source>

Kamu dapat menentukan teks di dalam element <video> yang akan ditampilkan sebagai cadangan di browser yang tidak mendukung penyematan video.

Menyematkan video di halaman web anda saat tidak diperlukan akan mengganggu pengunjung khususnya jika mereka secara otomatis bermain.

Atribut Video HTML

  1. auotplay: jika ditentukan, video yang disematkan akan secara otomatis mulai diputar segera setelah dapat melakukannya;
  2. controls: jika ditentukan, video yang disematkan akan memiliki controls yang disediakan oleh ketinggian;
  3. height: browser menentukan ketinggian video yang disematkan area tampilan dalam piksel;
  4. loop: jika ditentukan, video yang disematkan diputar berulang kali dalam loop tanpa akhir;
  5. mute: jika ditentukan, video tersemat akan dibisukan;
  6. poster menentukan jalur file yang mengarah ke file gambar yang akan ditampilkan saat video sedangn diunduh;
  7. src mennetukan jalur file mengarah ke file video;
  8. width: menentukan lebar area tampilan video tersemat dalam piksel.

Attribut autoplay, controls, loop dan muted adalah attribut boolean yang tidak memerulkan nilai. Mereka hanya perlu hadir di tag awal.

Attribut autoplay contoh

<video src="video.mp4" width="480" height="320" autoplay>support</video>

Attribute controls contoh

<video src="video.mp4" width="480" height="320" controls>support</video>

Attribute loop contoh

<video src="video.mp4" width="480" height="320" loop>support</video>

Attribute muted contoh

<video src="video.mp4" width="480" height="320" muted>support</video>

Jika attribute autoplay dan controls tidak ada di tag awal, video akan diputar dalam HTML.

Menggunakan element <source>

Beberapa browser tidak mendukung semua format video, solusinya adalah menyediakan banyak file menggunakan element <source>.

Contoh

<video controls>

<source src="video.mp3" type="video/mp3">

<source src="video.avi" type="video/avi">
