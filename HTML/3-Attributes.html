Attributes HTML

Attribut HTML dapat digunakan untuk menambahhkan lebih banyak informasi element.

Element penting HTML

  1. Attribut HTML ditemukan dalam tag HTML.
  2. Attribut HTML hanya muncul di awal tag. Itu tidak akan pernah ada di tag akhir.
  3. Element HTML dapat memiliki banyak attribut.
  4. Attribut HTML terdiri dari pasangan nama/ nilai.
  5. Ada beberapa attribut yang bisa digunakan pada semua HTML, Diingat meskipun attribut tersebut mungkin tidak berpengaruh pada beberapa element. Mereka disebut Attribut Global.

Attribut HTML terdiri dari

  1. Nama Attribut
  2. Tanda sama dengan (=)
  3. Nilai yang diapit oleh tanda kutip situasi

Ini terlihat seperti ini : attributename="value"

Kamu juga dapat menggunakan tanda kutip tunggal tergantung pada "nilai" esp. ketika nilainya mengandung tanda kutip ganda. hanya akan mengguanakan tanda kutip ganda.

Dalam pelajaran ini kita akan mempelajari beberapa attribut HTML dengan contoh.

Attribut lang Contoh:

                <!DOCTYPE html>
                <html lang="en-US">
                <!-- Isi dokumen html ada disini -->
                . . . 
                <html>

Kami menggunakan attribut lang untuk menentukan bahasa file HTML yang didefinisikan di atas adalah bahasa inggris America.

Attribut href Contoh:

<a href="http://www.example.com"> Goo to example </a>

Tautan ditentukan menggunakan element <a> jangkar.

Pada contoh diatas kami menggunakan attribut href untuk memberi tahu browser ke mana harus pergi.

Saat diklik, pengguna akan dialihkan ke http://www.example.com.

Contoh judul Attribut

<a href="#link" title="Saya melayani sebagai tooltip">Link</a>

Output

Link

Attribut judul menyediakan tooltip untuk Element HTML.

Sayangnya, itu tidak berfungsi pada perangkat seluler.

Contoh Gaya attribute

<p style="font-size: 40px; color: gold">

Saya adalah paragraf dengan ukuran 40 piksel dan dengan warna emas.

</p>

Output

Saya adalah paragraf dengan ukuran 40 piksel dengan warna emas.

Pada contoh yang diberikan di atas kita telah membuat sebuah paragraf menggunakan element.

<p> Kami juga mengguanakn attribut style untuk mengubah font-size dan color.

Attribut id dan class Contoh:

                <div class="name">
                    <!-- Kontent ada disini -->
                </div>

                <div id="name">
                    <!-- Konten ada disini -->
                </div>

Attribut id dan class memberikan referensi ke element didalamnya sebuah dokumen HTML.

Beberapa element dapat memiliki nilai/ nama kelas yang sama.

Nilai id harus unik untuk setiap element.

Ini membantu kami memilih element dalam style sheet dan script.
