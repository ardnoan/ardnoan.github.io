Entites Karakter HTML

Beberapa karakter dicadangkan dalam HTML. Misalnya jika kamu menulis tanda kurang dari < dan tanda lebih besar di browser > browser dapat mencampurinya dengan tag.

Dalam situasi seperti itu kita harus menggunakan karakter HTML entites.

Ada tiga cara untuk mendefinisikan entities karakter HTML.

  1. &nama kesatuan;. Entites HTML bernama encoding.
  2. &#entity_number; pengkodean desimal entities HTML.
  3. &#xentity_hexadecimal; heksadesimal entities HTML pengkodean.
