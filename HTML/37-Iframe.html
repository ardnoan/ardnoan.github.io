Iframe HTML

Iframe HTML juga dikenal sebagai element HTML Inline Frame mewakili konteks penjelajahan bersarang karena menyematkan halaman HTML lain ke halaman saat ini.

Itu dapat didefinisikan menggunakan element <iframe>.

Konten penjelajahan tersemat yang dibuat oleh element <iframe> itu sendiri merupakan lingkungan dokumen yang lengkap sehingga memiliki riwayat sesinya sendiri.

Attribut Iframe HTML

  1. height: menentukan tinggi bingkai dalam piksel;
  2. nama: jenis nama untuk iframe yang dapat digunakan sebagai nilai attribut target element seperti <a></a> dan <form></form>;
  3. src: menentukan URL atau jalur file dari halaman yang akan disematkan;
  4. width: menentukan lebar bingkai dalam piksel.

Contoh Attribut tinggi dan lebar dan src

<iframe src="iframe.html" height="300" width="400"></iframe>

Contoh attribute name example

<form action="contoh.html" target="demo">

<input type="submit">

</form>

<iframe name="demo" height="300" width="400"></iframe>

Contoh embedding ekternal web site

<iframe src="https://example.com" height="300" width="400"></iframe>

Contoh removing HTML iframe border

<iframe src="iframe.html" height="500" width="400" style="border: 0px;"></iframe>
