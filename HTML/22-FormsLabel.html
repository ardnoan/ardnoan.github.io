Forms Label

Label Formulir adalah element sebaris. Ini dapat dikaitkan dengan element formulir atau kontrol formulir seperti input, textarea, dan select untuk tujuan aksebilitas.

Element <label> mendefinisikan label bentuk HTML.

Untuk mengaitkan label ke input, kita dapat menggunakan attribut for dengan nilai yang sama dengan attribut id input.

Input

<label for="demo"> saya adalah label:

<input type="text" id="deno">

Output

Seperti yang telah disebutkan, label formulir adalah untuk tujuan aksesibilitas coba klik label untuk menfokuskan teks bidang.

Cara lain untuk mengasosiasikan label ke input adalah menyarangkan input di dalam element <label>.

Contoh

<label>Saya label:

<input type="text">

</label>

Output

Styling label bentuk HTML

<style type="text/css">

label {

border: 1px inset blue;

border-radius: 5px;

background: yellow;

color: navy;

padding-left: 4px;

padding-right: 4px;

margin-right: 3px;

}

</style>

<label for="demo"> nama awal:

<input type="text" id="demo"

Akan ada lebih banyak yang keren untuk label formulit HTML berikutnya jadi lanjutkan, nikmati dan rasakann.
