Attribute Element Formulir

Attribute yang dapat digunakan pada sebagian besar Element formulir seperti input, textarea, dan pilih (kontrol formulir).

  1. autocapitalize;
  2. autocomplete;
  3. autofocus;
  4. disabled;
  5. max;
  6. maxlength;
  7. min;
  8. minlength.

Attribute autofocus dan disabled adalah attribut boolean yang tidak memerlukan nilai, mereka hanya perlu hadir di tag awal.

Attributkapitalisasi otomatis (autocapitalize)

Ini menunjukan apakah nilai teks harus dikapitalisasi secara otomatis saat diimasukan/ diedit oleh pengguna.

<label>

the sentences value:

<input type="text" autocapitalize="sentences">

</label>

<label>

the words value:

<input type="text" autocapitalize="words">

</label>

<label>

the characters value:

<input type="text" autocapitalize="characters">

</label>

<label>

turning autocapitalize off:

<input type="text" autocapitalize="off">

</label>

Attribute autocomplete

Ini menunjukan apakah nilai teks harus diselesaikan secara otomatis oleh browser.

<label>

turning autocomplete on:

<input type="text" autocapitalize="on">

</label>

<label>

turning autocapitalize off:

<input type="text" autocapitalize="off">

</label>

Attribute auto focus

Ini menentukan apakah control formulir harus difokuskan sebagai halaman web.

<input type="text" autofocus>

Attriubte disabled

Ini menunjukan bahwa pengguna tidak dapat berinteraksi dengan kontrol formulir. Itu juga membuat control tidak adapat diklik dan dipilih.

<input type="text" value="I am disabled." disabled>

<textarea disabled> I am disabled </textarea>

<select disabled>

<option value="Option 1"> option 1 </option>

<option value="Option 2"> option 2 </option>

</select>

Attribut maxlength

Ini menentukan jumlah karakter maksimum yang dapat dimasukan pengguna.

<label>

The input below only allows a maximum of five characters. Try to type "loremipsm" (without space) then press the space button.

<input type="text" maxlength="5">

</label>

Attribut min

Ini menentukan nilai minimum untuk control formulir. INi adalah attribut HTML5.

<form action="contoh.html">

<input type="date" min="2000-02-02">

<input type="number" min="5">

<input type="submit"

Attribut minlength

Ini mentukan jumlah karakter minimum yang dapt digunakan pengguna memasuki.

<form action="contoh.html">

<input type="text" minlength="5">

<input type="submit">

Attribut banyak

Ini menunjukan apakah beberapa item dapt dipilih dalam kontrol formulir. Ini adalah attribut HTML5.

<p>The user can now select multiple files (selecting multiple files in a single input may not be possible on some android devices).</p>

<input type="file" multiple>

Attribute name

Ini menentukan nama kontrol yang mewakili datanya saat dikirimkan ke server.

<p>after submitting the form below the page handler can get the value of the input by referring to its name. </p>

<form action="contoh.html">

<input type="text" name="demo">

<input type="submit">

</input>

Attribute pattern

Ini menentukan ekspresi reguler yang nilai inputnya diperiksa melawan. ini adalah attribut HTML5.

<p>Try to fill the text field with a non-alphanumeric value then submit the form. we can use the title attribute to describe the paa

<form action="contoh.html">

<input type="text"pattern="[a-zA-Z0-9]+" title="Alphanumeric values only">

<input type="submit">

</input>

Attribute placeholder

Ini menentukan petunjuk kepada pengguna tentang apa yang harus dimasukan pada bidang teks.

<input type="text" placeholder="Halo saya placeholder alias teks transfarant">

Attribute readonly

Inin menunjukan bahwa pengguna tidak dapat memodifikasi kontrol tetapi tidak larang mengklik dan memilih.

<input type="text" value="I can only be read and selected but not be modified." readonly>

<textarea readonly> I can only be read and selected but not be modified.</textarea>

Attribute required

Ini menunjukan bahwa control formulir harus diisi sebelum mengirimkan formulir. Ini adalah attribut HTML5.

<p>Try submitting the form belowwithout filling up the text field. </p>

<form action="contoh.html">

<input type="text" required>

<input type="submit">

Attribute spellcheck

Ini menunjukan apakah bidang teks harus memiliki ejaan dan tata bahasa diperiksa.

<label>

The default behavior:

<textarea spellcheck="default"></textarea>

<br>

<label>

Turning spellcheck on:

<textarea spellcheck="true"></textarea>

<br>

<label>

Turning spellcheck off:

<textarea spellcheck="false"></textarea>

<br>

>!-- we need the cols attribute to take effect -->

</label>

<label>

The soft value:

</label>
