Daftar tarik turun HTML

Daftar dropdown HTMl adalah kontrol yang menyediakan menu pilihan. Ini adalah bentuk dari element inline.

Element <select> mendefinisikan daftar dropdown HTML. itu harus menyertakan beberapa element <option> dengan attribut value untuk masing-masing.

Attribut tarik turun HTML

Multiple menentukan apakah beberapa pilihan dapat dipilih dari daftar dropdown, attribut ini tidak memerlukan nilai, hanya perlu ada di tag awal.

Dengan javascript kita dapat dengan mudah mendapatkan nilai dari element <select>, untuk melihat bagaimana melakukannya.

Contoh

<select id="percobaan">

<option value="apel">apel</option>

<option value="jeruk">jeruk</option>

<option value="anggur">anggur</option>

<option value="pisang">pisang</option>

<option value="stroberi">stroberi</option>

</select>

<button onclick="ambilNilai();">ambil nilai</button>

<script>

function ambilNilai() {

var nilai = document.getElementById('percobaan').value;/p>

alert(nilai);

}

</script>

Multiple Options allowed contoh :

<select id="percobaan" multiple>

<option value="apel">apel</option>

<option value="jeruk">jeruk</option>

<option value="anggur">anggur</option>

<option value="pisang">pisang</option>

<option value="stroberi">stroberi</option>

</select>

<button onclick="ambilNilai();">ambil nilai</button>

<script>

function ambilNilai() {

var pengaturan = document.getElementById('percobaan').options;

var pengaturan_nilai = document.getElementById('percobaan').options.length;

var nilai = [];

for(var i = 0; i < pengaturan_nilai; i++) {

if(pengaturan[i].selected) {

nilai.push(pengaturan[i].value);

}

}

alert(JSON.stringify(nilai));

alert(nilai);

}

</script>
