Audio HTML

Untuk menyematkan audio dalam HTML, kita harus menggunakan element <audio> dengan attribut src atau element <source> dengan attribu src dan type untuk masing-masing.

Element <source> akan dijelaskan dibawah ini.

Kamu dapat menentukan teks didalam element <audio> yang akan ditampilkan sebagai cadangan di browser yang tidak mendukung penyematan audio.

Peringatan! menyematkan audio di halaman web anda jika tidak diperlukan akan mengganggu pengunjung khusunya jika mereka secara otomatis bermain.

Attribut audio HTML

  1. autoplay: jika ditnetukan, audio yang disematkan akan melakukannya secara otomatis mulai diputar segera setelah dapat melakukannya
  2. controls: jika ditentukan, audio yang disematkan akan memilikinya kontrol yang disediakan oleh browser.
  3. loop: jika ditentukan audio yang disematkan akan diputar adan lagi dalam putaran tanpa akhir
  4. muted: jika ditentukan audio yang disematkan akan dinonaktifkan
  5. src: menetukan jalur file yang mengarah ke file audio.

Attribut autoplay, controls, loop dan muted adalah attribut boolean yang tidak memerlukan nilai. Meraka hanya perlu hadir di tag awal.

Attribute autoplay

<audio src="audio.mp3" autoplay> support?</audio>

Attribute controls

<audio src="audio.mp3" controls> support?</audio>

Attribut loop

<audio src="audio.mp3" autoplay loop> support?</audio>

Attribute muted

<audio src="audio.mp3" controls muted> support?</audio>

Peringatan! Jika attribut auotoplay dan controls tidak ada di tag awal, audio tidak akan diputar dalam HTML.

Menggunakan element <source>

Beberapa browser tidak mendukung semua format audio. solusinya adalah menyediakan banyak file menggunakan element <source>.

Contoh

<audio control>

<source src="audio.mp3" type="audio/mp3">

<source src="audio.ogg" type="audio/ogg">

Support?

</audio>

Browser akan menggunakan yang pertama didukungnya. Outputnya akan sama dengan contoh attribut controls di atas.
