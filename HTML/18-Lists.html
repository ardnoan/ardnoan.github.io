List HTML

Element dibawah mewakili list HTML yang dapat digunakan untuk mengelompokkan kumpulan item dengan atau tanpa urutan.

Element List HTML

  1. <ul> mendefinisikan unored list dimana urutannya tidak berarti dan biasanya berpoin;
  2. <ol> mendefinisikan list yang diurutkan dimana urutannya bermakna dan biasanya diberi nomor;
  3. <li> element turunan dari element <ul> dan <ol> yang mendefinisikan item daftar/ list.

Contoh :

                <!-- list tanpa urutan -->
                <p> buah </p>
                <ul>
                    <li> apel </li>
                    <li> jeruk </li>
                    <li> mangga </li>
                </ul>

                <!-- list terurut -->
                <ol>
                    <li> apel </li>
                    <li> jeruk </li>
                    <li> mangga </li>
                </ol>

Outuput:

buah

  • apel
  • jeruk
  • mangga

buah

  1. apel
  2. jeruk
  3. mangga
