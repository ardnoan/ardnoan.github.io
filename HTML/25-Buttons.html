Tombol HTML

Tombol HTML adalah element yang dapat diklik yang dapat digunakan dalam formulir HTML atau dimana saja dalam dokumen yang memerlukan fungsionalitas tombol standar.

Element <button> mendefinisikan tombol HTML.

Memanggil fungsi javascript (JS)

Kita dapat memanggil fungsi JS saat tombol diklik dengan menentukan atribut onclick dengan nilai yang setara dengan nama fungsi.

Kamu dapat melihat kembali materi javascript untuk memperjelas.

Jenis attribut

  1. type="submit" nilai default. Itu menyebabkan bentuknya menjadi ditampilkan.
  2. type="button" ini menghilangkan perilaku default tombol.
  3. type="reset" ini mengatur ulang kontrol formulir ke awal nilai.

Tombol HTML dalam formulir HTML

Mengklik tombol di dalam formulir HTML akan menyebabkan formulir dikirim karena nilai default diatur ke type="submit"

<form action="contoh-php.html">

<input type="text">

<button>submit</button>

</form>

Mengatur ulang nilai kontrol formulit menggunakan tombol Contoh

<form action="contoh-php.html">

<label> nama :

<input type="text">

</label>

<label> hobi :

<input type="text">

</label>

<button type="reset"> reset value </button>

<button type="submit"> submit </button>
