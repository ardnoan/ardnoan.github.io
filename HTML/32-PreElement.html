HTML Pre Element

Element HTML <pre> mewakili teks yang telah diformat sebelumnya yang harus dirender persis oleh browser seperti yang tertulis di file HTML.

Contoh

<p>

My

name

is

name.

</p>

<pre>

My

name

is

my

name.

<pre>
