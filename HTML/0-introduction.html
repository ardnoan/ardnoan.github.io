Hyper Text Markup Language (HTML)

HTML adalah singkatan dari Hyper Text Markup Language. HTML5 adalah versi utama saat ini yang menjadi standar pembuatan website.

Sangat penting untuk mempelajari HTML jika ingin membuat website. jika tidak tahu HTML maka tidak bisa membuat website oleh karena itu adalah salah satu syarat dalam mempelajari bahasa lain yang digunakan untuk pengembangan web.

Agar belajar lebih cepat maka lanjutkan untuk membacanya kemudian nikmati segala prosesnya. Belajar pemograman atau ngoding biasakan belajarnya disamakan dengan bermain lalu berseksperien lah sesuka hatimu.

Input

                <!DOCTYPE html>
                <html>
                <head>
                    <title> Bermain </title>
                </head>
                <body>
                    <h1> Brother, Sehat? </h1>
                    <p> Bagaimana keadaanya disana? </p>
                </body>
                </html>

Output

Brother, Sehat?

Bagaimana keadaanya disana?

Arti

  1. <!DOCTYPE html> artinya mendeklarasikan jenis dokumen atau versi HTML5 <html> element ini menmbungkus semua yang ada didalam dokumen html itu.
  2. <head> ini dapat mencakup script, tag dan banyak lagi. Elemen ini mendefinisikan judul halaman web akan muncul dibagian atas browser.
  3. <body> element ini menyelubungi elemen seperti <h1>, <p> dan masih banyak lagi.
  4. <h1> elemen ini mendefinisikan heading. <p> elemen ini mendefinisikan sebuah paragraf.

HTML Tags

Tag HTML adalah nama elemen yang dikelilingi oleh kurung sudut, dalam HTML kita memulai dan mengakhiri tag.

<p>Sampurasun.<p>

Tag awal dan Tag Akhir. Awal tag disebut "membuka tag" contoh <p>. Tag Akhir disebut "mengakhiri tag" contoh </p>

Ini adalah struktur dasar dari setiap halaman HTML. 

                <!DOCTYPE html>
                <html>
                <head>
                    <title> Judul </title>
                </head>
                <body>
                </body>
                </html>
