Meta HTML

Element HTML <meta> mewakiliki metadata itu. Harus selalu berada di dalam element <head>. Ini adalah element kosong.

Attribut http-equiv

Ini mendefinisikan instruksi yang dapat mengubah perilaku server dan agen pengguna. Nilai instruksi ditentukan dalam attribut konten.

Name attribut

Attribut ini menentukan nama bagian dari metadata tingkat dokumen. ini terkait dengan nilai attribut konten.

Content attribut

Nilai attribut ini berisi nilai untuk http-equiv atau attribut nama.

Membuat halaman web responsive dengan HTML meta

<meta name="viewport" content="width=device-width, initiala-scale=1.0"/>

Search Engine Optimization (SEO) dengan Meta HTML

Element Meta HTML adalah alat yang hebat untuk SEO, ini dapat membantu mesin telusur mengetahui tentang apa halaman itu sehingga dapat di indeks dan dicari oleh orang-orang.

<meta name="author" content="asep" />

<meta name="title" content="Judul yang ditampilkan dimesin pencari seperti google">

<meta name="description" content="Deskripsi ditampilkan dibawah judul pada ppencarian search engine google">

<meta name="keywords" content="htmnl, html tutorial, belajar html">
