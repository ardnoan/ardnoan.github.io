Paragraf HTML

Paragraf HTML terutama digunakan untuk mengelompokan kalimat. Element <p> mendefinisikan Paragraf HTML paragraf HTML adalah element tingkat block.

Contoh

<p> ini adalah paragraf </p>

Output

ini adalah paragraf

Menyelaraskan Paragraf HTML

Untuk menyelaraskan paragraf, kita perlu menggunakan properti CSS teks dengan nilai kiri, tengah atau kanan.

Contoh

<p style="text-align: left;"> paragraf rata kiri </p>

<p style="text-align: center;"> paragraf rata tengah </p>

<p style="text-align: right;"> paragraf rata kanan </p>

Output

paragraf rata kiri

paragraf rata tengah

paragraf rata kanan

Menata Paragraf HTML

Contoh dibawah akan menggunakan gaya inline untuk gaya paragraf.

Contoh

<!-- ubah warna -->

<p style="color: gold"> warna emas </p>


<!-- mengubah ukuran font -->

<p style="font-size: 25px;"> ukuran font adalah 25px </p>


<!-- mengubah jenis font -->

<p style="font-family: verdana;"> font family: verdana </p>


<!-- mengubah bobot font -->

<p style="font-weight: bold;"> teks yang dicetak tebal </p>

Output

warna emas

ukuran font adalah 25px

font-family: verdana

teks yang dicetak tebal
