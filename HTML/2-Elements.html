Elements HTML

Pada pelajaran pertama kita telah mempelajari tag dan hal-hal seperti start tag (membuka tag) dan tag akhir (menutup tag).

Elements HTML, biasanya terdiri dari "Tag awal", "Konten", dan "Tag Akhir".

Contoh

<p> konten element </p>

Output

konten element

Contoh Element HTML di atas terdiri dari :

  1. Tag Awal <p>
  2. Kontent Element " kontent element "
  3. Tag Akhir </p>

Contoh

<h1> Mempelajari Element HTML </p>

Output

Mempelajari Element HTML

Contoh element HTML di atas terdiri dari

  1. Tag Awal <h1>
  2. Kontent Element " Mempelajari Element HTML "
  3. Tag Akhir </h1>

Element HTML Bersarang

Ada beberapa kasus dimana element HTML dapat berisi satu atau lebih element HTML.

Agar lebih memahaminya lihatlah contoh kode dibawah ini.

<p><i> Teks Miring </i></p>

Output

Teks Miring

Contoh element HTML di atas terdiri dari

  1. Mulai Tag <p>
  2. Mulai TAg <i>
  3. Kontent element. "Teks Miring"
  4. Akhir Tag </i>
  5. Akhir Tag </p>

Pada contoh diatas, ada 2 tag awal dan 2 tag akhir tag kedua yaitu <i> mencetak miring teks didalamnya.

Element Kosong

Element Kosong adalah element yang tidak memiliki konten element dan tag akhir.

Daftar tag akhir.

  1. <meta />
  2. <link />
  3. <img />
  4. <br />
  5. <hr />
  6. <input />

Praktik terbaik dalam HTML Empty Elements adalah selalu mengedepankan garis miring atau tanda sebelum tanda lebih besar dari >.

Dengan cara ini mereka ditutup pada tag awal mereka.
