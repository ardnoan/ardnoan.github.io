JavaScript HTML

JavaScript (JS) adalah bahasa scripting yang digunakan untuk membuat halaman web dinamis dan lebih interaktif.

JS pada umumnya digunakan untuk validasi form, kontent dinamis, efek dan manipulasi gambar.

JavaScript dalam HTML

Untuk menggunakan JS dalam HTML, kita dapat melakukan salah satu dari berikut ini.

  1. Gunakan element <script type="text/javascript"> dengan attribut type yang menentukan JS dan letakkan pernyataan skrip didalamnya.
  2. Gunakan attribut src dari element <script> yang menunjuk ke file JS lain, misalnya. <script src="script.js"></script>

Ekstensi file JS adalah ".js" Hanya menulis pernyataaan skrip di dalam file JS jangan letakkan <script></script> element di dalamnya.

Praktik terbaik adalah meletakan element <script> sebelum akhir tag element <body>

Struktur dokument HTML dengan contoh Javascript

<!DOCTYPE html>

<html>

<head>

<title></title>

</head>

<body>

<script type="text/javascript">

<!-- semua pernyataan scripting ada disini -->

</script

</body>

</html>

kita akan menggunakan metode document.getElementById("id") untuk memilih element HTML tertentu dan alert(string) untuk menampilkan kotak.

Berikut adalah beberapa contoh sederhana penggunaan javascript dalam HTML.

Contoh Kotak Peringatan

<script type="text/javascript">

Alert("Halo Saya sebuah kotak peringatan");

</script>

Dynamic Content contoh

<p id="sampel"> halo dunia </p>

<button onclick="changeContent();"> Pilih content </button>

<script type="text/javascript">

function changeContent() {

document.getElementByID("sampel").innerHTML = "Lorem ipsum";

}

Dynamic style contoh

<p id="sampel"> halo dunia </p>

<button onclick="changeStyles();> Pilih style </button>

<script type="text/javascript">

function changeStyles() {

document.getElementById("sampel").style.fontSize = "20px";

document.getElementById("sampel").style.fonStyle = "italic";

document.getElementById("sampel").style.color = "gold";

}

Hiding and Showing contoh

<div id="sampel"> style="width: 200px; height: 200px; background-color: gold; border: 5px solid black;></div>

<button onclick="show()> show </button>

<button onclick="hide()> hide </button>

<script type="text/javascript>

function show() {

document.getElementById("sampel").style.display = "block";

}

function hide() {

document.getElementById("sampel").style.display = "none";

}

Dyanmic image contoh

<img src="images.jpg" style="max-width: 100%" id="sampel" class="1"

<br/>

<button onclick="changeImage();> change image </button>

<script type="text/javascript">

function changeImage() {

document.getElementById("sampel").src = "images1.jpg";

}
