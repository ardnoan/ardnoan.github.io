HTML5 Sections

Bagian HTML mengelompokkan element HTML yang berbeda (contohnya teks, judul, paragraf, gambar, dll).

Membuat bagian-bagian tertentu sangat penting untuk mengatur dan menata web isi.

Element yang digunakan untuk bagian HTML

  1. <div> digunakan untuk mengelompokan sekelompok besar element HTML seperti navigasi, header, kontent utama, footer, gambar. itu adalah element tingkat blok.
  2. <span> digunakan untuk mengelompokan kelompok teks yang lebih kecil dalam sebuah paragraf dan beberapa element HTML. itu adalah element sebaris

Element Semantik HTML5

Berikut adalah daftar element semantik HTML yang berfungsi serupa dengan element <div>.

  1. <nav> menentukan daftar atau bilah navigasi
  2. <header> mendefinisikan header
  3. <main> mendefinisikan kontent utaqma
  4. <footer> mendeinisikan footer

Menggunakan elemnt sematik HTML5 ini tidak wajib tetapi merupakan praktik yang baik dan yang disarankan karena dapat membantu dalam Serach Engine Optimization (SEO) dan kode kamu akan lebih mudah dipahami.

Namun elemnt ini bekerja dengan cara yang sama dengan element <div> dalam hal rendering di browser.

<div> Contoh:

<!-- mari buat div dengan kontent dan gaya yang berbeda -->

<div style="background: gold; width: 400px; height: 400px;>

<p> Ini adalah sebuah paragraf

</div>


<div style="background: yellow; width: 400px; height: 400px;">

<p> Ini adalah sebuah paragraf </p>

</div>

Seperti yang kalian lihat, dua element <div> dipisahkan satu sama lain, mereka juga memiliki gaya yang berbeda. Itulah salah satu keuntungan bagian HTML, ini merupakan kita menata bagian tertentu dari dokument HTML.

<span> Contoh :

<!-- Mari kita letakan beberapa teks didalam span dan style -->

<p>

<span style="font-size: 30px;>I</span> <span style="color: blue">saya sebuah paragraf</span> <span style="color: green:>paragraf</span> <span style="color: red">Paragraf</span>

</p>

Output

Isaya sebuah paragraf paragraf Paragraf

Seperti yang kalian lihat, element <span> tidak memisahkan paragraf menjadi baris.

Selalu ingat. Element <span> bagus untuk menata dan mengatur beberapa element dan element sebaris sedangkan element <div> bagus untuk element yang lebih besar.
